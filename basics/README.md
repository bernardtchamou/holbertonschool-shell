0-current_working_directory
indique la commande permettant de montrer le chemin actuel

1-listit
commande qui liste le contenu du dossier actuel

2-bring_me_home
commande qui change de dossier

3-listfiles
commande qui liste le contenu d'un dossier en format long

4-listmorefiles
commande qui liste le contenu d'un dossier en format long, y compris les fichiers cachés

5-listfilesdigitonly
commande qui liste le contenu d'un commande qui liste le contenu d'un dossier en format long, y compris les fichiers cachés et les identifiants numériques

6-firstdirectory
commande qui crée un dossier/répertoire "my_first_directory" dans le dossier /tmp

7-movethatfile
commande qui déplace le fichier "betty" du dossier "tmp" vers le fichier "my_first_directory" du dossier /tmp

8-firstdelete
commande qui supprime le fichier "betty" du dossier "my_first_directory", lui même présent dans le dossier /tmp

9-firstdirdeletion
commande qui supprime le répertoire "my_first_directory" dans le répertoire /tmp

10-back
commande qui change de dossier/répertoire, du dossier actuel vers le dossier précédent, dans le temps

11-lists
commande qui liste tous les fichiers en long format, y compris les fichier cachés, dans le dossier actuel, le dossier précédant et le dossier /boot, dans cet ordre

12-file_type
commande qui montre le type du fichier "iamafile", présent dans le dossier /tmp

13-symbolic_link
commande qui crée un lien nommé __ls__ qui pointe vers le fichier "ls" dans le dossier /bin

14-copy_html
commande qui copie tous les fichiers finissant par ".html" du dossier actuel vers le dossier parent

15-lets_move
commande qui déplace tous les fichiers commançant par une majuscule dans le dossier /tmp/u

16-clean_emacs
commande qui supprimme tous les fichiers du répertoire actuel finissant par ~

17-tree
commande qui crée des répertoires en arborescence où le répertoire "welcome" est compris dans le répertoire "to", lui même compris dans le répertoire "school"
